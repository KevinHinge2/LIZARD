# Tom — Premium Lizard Launch Page

A polished, single-page parody product launch site for Tom pressing a red button.

## Run locally

Because this is a static site, you can:

1. Open `index.html` directly in your browser, or
2. Serve the folder with a tiny local server:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Swap in your assets

- **Tom media**: Replace `assets/tom-placeholder.svg` in `index.html` with your image/video/GIF path.
- **Lizard sound**: Replace `assets/lizard-sound.mp3` in `index.html` with your real sound file.

Look for inline comments in `index.html` for exact replacement locations.
